목적
 - REST API 사용하여 앱과 서버의 통신

사용 언어
 - REST API를 사용하기 위한 Nest.js
   - Nest.js를 사용하기 위한 Node.js, npm

사용 툴
 - 코드 작성을 위한 Visual Studio, Intelli J

사용 형식
 - JSON, XML 중에서 JSON으로 통일.

관련 Open API 주소
 - 카카오톡 : https://developers.kakao.com/docs/latest/ko/kakaologin/rest-api

리소스 정리
 - Collections : http://example.com/topics/1에서 topics값. '복수'
 - Element : http://example.com/topics/1에서 1, 보통 ID값. rest 등으로 사용도 가능. '단수'

기능(메소드) 정리
 - 4가지가 전부 : 생성, 읽기, 갱신(수정), 삭제 (끝).
   - 생성 : Create / post (REST 기준 / HTTP or Git 기준)
   - 읽기 : Read / get
   - 수정 : Update / put | patch
   - 삭제 : Delect / delete
